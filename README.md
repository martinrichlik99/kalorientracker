# Vitality – Kalorientracker

Persönliche Kalorien- und Makro-Tracking-PWA. Single-user, offline-fähig, installierbar.

- **Stack:** Vanilla HTML/CSS/JS + Tailwind (CDN) + Service Worker
- **Speicher:** localStorage + IndexedDB (lokal am Gerät)
- **Food-DB:** OpenFoodFacts (Suche + Barcode via ZXing)
- **Design:** Vitality Minimalist (Emerald `#10b981`, Hanken Grotesk)

## Nutzung
Öffnen → „Zum Home-Bildschirm hinzufügen" → läuft wie native App.
